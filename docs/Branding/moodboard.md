# Moodboard — RentAI

## Concepto Visual

La identidad de RentAI combina **calidez residencial** con **tecnología moderna**. El diseño evoca confianza, comunidad y frescura.

## Paleta de Colores

Ver carpeta `colores/` para referencias visuales.

| Nombre | Hex | Uso |
|---|---|---|
| Verde principal | `#63A694` | Acciones primarias, arrendatario |
| Verde claro | `#A8D1B1` | Fondos, onboarding |
| Terracota | `#D87D6F` | Propietario, acciones de aceptación |
| Café suave | `#82554D` | Textos secundarios |
| Negro | `#0D0D0D` | Texto principal |

## Tipografía

Ver carpeta `fonts/` para archivos de fuentes.

- **Títulos:** Sans-serif bold, tracking tighter — transmite solidez y modernidad
- **Cuerpo:** Sans-serif regular — legibilidad en móvil y escritorio
- **Labels UI:** Uppercase, letter-spacing wide — jerarquía clara

## Estilo de Ilustración

- Tarjetas con bordes redondeados (`rounded-2xl`, `rounded-3xl`)
- Sombras suaves (`shadow-lg`, `shadow-xl`)
- Avatares circulares con inicial de nombre
- Iconografía de Lucide React

## Referencias de Diseño

- Inspiración en apps de matching (Tinder, Bumble) para el flujo de swipe
- UI limpia tipo fintech para las secciones de propietario
- Animaciones con Framer Motion — transiciones suaves y naturales
