# Tipografía — RentAI

## Fuentes utilizadas

- **Sans-serif del sistema** (via Tailwind) — cuerpo y UI general
- Pesos usados: `font-medium`, `font-semibold`, `font-bold`, `font-black`

## Escala tipográfica

| Elemento | Clase Tailwind | Uso |
|---|---|---|
| Título hero | `text-3xl md:text-4xl font-bold` | Onboarding, headers |
| Subtítulo | `text-xl font-semibold` | Secciones |
| Label UI | `text-[11px] font-black uppercase tracking-widest` | Campos de formulario |
| Cuerpo | `text-sm font-medium` | Descripciones |
| Caption | `text-xs text-muted-foreground` | Fechas, metadata |
